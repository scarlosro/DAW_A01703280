Cuestionario 1: Base de Datos

1.- ¿Qué es entonces una base de datos - DB?

Un reservorio de información útil para un usuario, empresa o institución, guardada de manera ordenana,una colección de datos relacionados entre sí y administrada por programas gestores de base de datos [1]

2.- ¿En qué casos es conveniente usar base de datos?

Cuándo se va a manejar distinta información por lo cual manejar archivos como excel, o escritos pueden causar muchos errores, por lo cual la base de datos simplificaría los procesos así como la redundancia.

3.- ¿Qué es un sistema de gestión de base de datos -DBMS?

Es un sistema gestor de base de datos es una aplicación que permite a los usuarios definir, crear y mantener una base de datos, y proporciona el acceso controlado a la misma. [2]

4.- Enlista y explica las funciones/responsabilidades que tienen los DBMS
  1. Inserción, actualización, eliminación y consulta de datos
  2. Permitir un acesso controlado a la base de datos
  3. Un sistema de integridad que mantiene la consistencia de los datos
  4. Un sistema de contro de concurrecia que permite el acceso compartido a la base de datos
  5. Un sistema de recuperación que restablece la base de datos después de que se produzca un fallo de hardware o del software [2]
5.- Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos

En en desarrollo de proyecto de desarrollo de videjuegos podríamos a ver creado una base de datos para poder permitir que se registren diferentes jugadores y puedan conservar su avance, nivel, puntaje, etc. Y poder hacer aún más escalable el juego.
