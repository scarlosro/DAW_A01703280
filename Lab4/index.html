Preguntas

1.- ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

JavaScript no es un lenguaje compilado, es interpretado solamente. JavaScript está basado en objetos, mientras que JAVA es un lenguaje orientado a Objetos, no utiliza clases, herencias o técnicas habituales en la OOP. No existe la declaración de tipos de variables a utilizar.[1]

5.- ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Permite el acceso desde cualquier función, así como su modificación de ella. Pero si se crea una variable local dentro de una función con el mismo nombre de una global, está prevalecerá. [4]

Se puede utilizar los métodos: match, search, replace, split. Se puede usar para buscar una cadena de texto regulares.[5]
